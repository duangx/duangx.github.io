rsync备份

作者：duangx      发布时间：2018-01-16      标签： rsync

全网备份案例

  1. 需要备份的文件或目录/var/spool/cron/root；/etc/rc.local；/etc/sysconfig/iptables；/var/www/html；/app/logs
  2. 每台web服务器备份到本地的/backup目录下以本机IP命名的目录，本地保留7天。
  3. 打包文件命需含有当天的日期，使用rsync定时推送到备份服务器。
  4. 检查备份服务器上是否正常备份，将结果发给管理员，每周六数据永久保留，其余保留180天。
